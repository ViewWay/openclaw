## 🔧 关键技术选型

### 国际化 (i18n)

- **框架**: i18next (成熟、功能强大)
- **加载**: 懒加载 + 缓存
- **检测**: 浏览器语言 + 配置文件
- **回退**: en → zh-CN → en

### 认证 (Authentication)

- **Token**: JWT + 过期时间
- **2FA**: TOTP (speakeasy)
- **存储**: 配置文件 + 内存缓存
- **加密**: bcrypt + crypto

### 权限 (Authorization)

- **模型**: RBAC + ACL 混合
- **存储**: JSON 配置 + 内存缓存
- **检查**: 中间件 + 装饰器
- **审计**: 文件日志 + 数据库（可选）

### 日志 (Logging)

- **格式**: JSON 结构化日志
- **输出**: 文件 + 控制台
- **过滤**: 敏感信息自动过滤
- **国际化**: 支持双语输出

## 📝 扩展性设计

### 添加新语言

1. 创建语言文件: `ui/src/i18n/locales/xx-XX.ts`
2. 添加到支持列表: `SUPPORTED_LOCALES`
3. 翻译所有文本键
4. 测试新语言

### 添加新权限

1. 定义权限类型: `Permission`
2. 添加权限映射: `RolePermissions`
3. 实现权限检查: `PermissionManager`
4. 添加审计日志: `PermissionAuditor`

### 添加新认证方式

1. 实现认证接口: `AuthProvider`
2. 添加配置项: `GatewayAuthConfig`
3. 集成到认证流程: `authorizeGatewayConnect`
4. 添加测试用例
